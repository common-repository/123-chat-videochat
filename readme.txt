== Installation ==

[youtube https://youtu.be/p9ltagMSVPs]

= EN: =
1. Register at www.123.chat > log in > click on "Settings" > "Wordpress Plugin" > copy user ID
1. Install the 123.chat plugin in WordPress backend (Plugins > Install > search for "123chat" > Install + activate)
1. In WordPress-backend go to > Settings > 123.chat > insert user ID > save
1. In 123.chat-backend go to > LiveChat > Test camera and sound > let's go!
1. The browser window can be minimized - when a web visitor requests a contact, a signal tone and a message window appear!

= DE: =
1. Bei www.123.chat anmelden > einloggen > auf "Einstellungen" klicken > "WordPress Plugin" > User-ID kopieren
1. Im WordPress-Backend das 123.chat-Plugin installieren (Plugins > Installieren > "123chat" suchen > Installieren + Aktivieren)
1. Im WordPress-Backend auf > Einstellungen > 123.chat > User-ID einfügen > speichern
1. Im 123.chat-Backend zu > LiveChat > Kamera und Ton testen > los geht's!
1. Das Browserfenster kann minimiert werden - bei Kontaktanfrage eines Web-Besuchers kommt ein Signalton und Hinweisfenster!
